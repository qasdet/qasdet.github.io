Что такое Playwright, его возможности, поддержка браузеров и языков.

Установка

Запуск тестов

Автономная библиотека

Действия

Клик, ввод текста, выбор опций, загрузка файлов и drag-and-drop.

Автоматическое ожидание

Принципы Auto-Waiting и Actionability Checks.

Локаторы

get_by_role, get_by_text, get_by_test_id и цепочки локаторов.

Другие локаторы

CSS с расширениями Playwright, XPath, текстовые и n-ные локаторы.

Утверждения

expect, soft-assertions, проверка видимости, текста, атрибутов.

События

Браузерные контексты

Изолированные сессии, настройка размера/языка/геолокации.

Страницы (Page)

Управление вкладками, popup-окнами и multi-page сценариями.

Фреймы

Работа с iframe, вложенные фреймы, content_frame.

ElementHandle

Низкоуровневая работа с DOM-элементами.

Тестирование API

APIRequestContext, GET/POST, проверка ответов и куки.

Мокирование

Перехват и подмена запросов, route, fulfil и abort.

Сеть

Мониторинг запросов, WebSocket, HTTPS-аутентификация.

Отладка

Playwright Inspector, логирование, pause(), пауза в ошибках.

Просмотрщик трассировок

Запись и анализ трассировок, Trace Viewer GUI.

Генератор тестов

codegen, запись действий, автогенерация локаторов.

Скриншоты

Полностраничные, элементов, в буфер, сравнение с эталоном.

Видео

Запись видео прохождения тестов.

ARIA-снимки

Snapshot-тестирование accessibility-дерева.

Эмуляция устройств

Мобильные устройства, viewport, геолокация, язык.

Аутентификация

Cookies, localStorage, storageState для повторного входа.

Расширения Chrome

WebView2

Сенсорные события

Clock API

Диалоги

Загрузки

Расширяемость

CI

Официальные образы, запуск в контейнерах.
