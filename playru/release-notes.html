Примечания к релизам

История версий Playwright для Python. Новые возможности, улучшения и изменения в каждом релизе.

v1.61

  • WebAuthn passkeys — новый Credentials API для тестирования аутентификации с помощью passkeys (WebAuthn). Позволяет создавать и использовать виртуальные учётные данные.
  • WebStorage API — методы local_storage и session_storage для прямого чтения/записи localStorage и sessionStorage на странице.
  • api_response.security_details() — получение деталей безопасности ответа (сертификат, протокол, шифр).
  • server_addr() — информация о сервере, обработавшем запрос.

v1.60

  • HAR-запись на трассировке — методы start_har / stop_har для записи HAR-файлов прямо в трассировке.
  • locator.drop() — симуляция drop файлов на элемент (drag-and-drop для файлов).
  • ARIA snapshot — метод page.aria_snapshot() для создания снимков accessibility-дерева.
  • BrowserContext lifecycle events — новые события для отслеживания жизненного цикла контекста.

v1.59

  • Screencast API — page.screencast() для записи экрана с аннотациями, оверлеями и захватом отдельных кадров.
  • browser.bind() — новый механизм интероперабельности между браузерами.
  • Dashboard CLI — инструмент командной строки для визуализации результатов тестового прогона.

v1.58

  • UI Mode / Trace Viewer — значительные улучшения интерфейса: тёмная и светлая тема, улучшенная навигация по трассировкам.
  • Улучшена производительность Trace Viewer при работе с большими трассировками.

v1.57

  • Chrome for Testing — Playwright перешёл на использование Chrome for Testing (CfT) вместо обычного Chrome. Это обеспечивает детерминированные версии браузера.
  • Удалён page.accessibility — устаревший API accessibility удалён из библиотеки.

v1.56

  • page.console_messages() — получение всех консольных сообщений страницы в виде списка.
  • page.page_errors() — получение всех ошибок страницы.
  • Улучшена обработка исключений в асинхронном API.

v1.55

  • Codegen with auto-visibility assertions — генератор тестов теперь автоматически добавляет проверки видимости элементов.
  • Сборка мусора в трассировке — оптимизация размера трассировок за счёт удаления неиспользуемых данных.

v1.54

  • WebSocket routing — route_web_socket() для перехвата и модификации WebSocket-соединений.
  • Новые методы locator: locator.clear(), locator.is_visible() с дополнительными опциями.

v1.53

  • Clock API — управление временем в тестах: page.clock.fast_forward(), page.clock.set_fixed_time() и другие методы.
  • Строгие локаторы по умолчанию — если локатор находит более одного элемента, Playwright выбрасывает ошибку. Поведение настраивается.

v1.52

  • TLS client certificates — поддержка клиентских TLS-сертификатов для аутентификации.
  • Новые события контекста: on_page, on_close, on_error.

v1.51

  • Locator highlights — подсветка локаторов в браузере при использовании Codegen.
  • Утверждение to_have_values для проверки значений в select-элементах.

v1.50

  • Accessibility assertions — to_match_aria_snapshot() для сравнения accessibility-дерева с эталоном.
  • Улучшена производительность при работе с большим количеством фреймов.

v1.49

  • ARIA snapshot — улучшения: поддержка вложенных элементов, более точное соответствие спецификации ARIA.

v1.48

  • Новые возможности Trace Viewer: фильтрация по типу события, поиск по локаторам, экспорт в JSON.
  • Codegen: поддержка записи действий в iframe.

v1.47

  • UI Mode Preview — предварительная версия графического интерфейса для запуска и отладки тестов.
  • Улучшена стабильность при параллельном запуске.

v1.46

  • Улучшения API тестирования: новые методы APIRequestContext, поддержка multipart-форм.
  • Оптимизация памяти при работе с большими страницами.

v1.45

  • Тестирование мобильных устройств — новая система эмуляции устройств с предустановленными профилями (iPhone, iPad, Pixel).
  • Поддержка сенсорных событий и жестов.

v1.44

  • Новые локаторы — get_by_placeholder, get_by_alt_text, улучшенный get_by_role.
  • Новые методы assert: to_be_in_viewport, to_contain_text с регулярными выражениями.

v1.43

  • Network mocking — полноценное мокирование сетевых запросов через browser_context.route() и page.route().
  • Эмуляция устройств — поддержка предустановленных устройств (iPhone 13, Pixel 6 и др.).

Примечание: Команда Playwright выпускает новый стабильный релиз примерно раз в месяц. Актуальная версия всегда доступна на PyPI и в официальном GitHub-репозитории.

Совет: Для обновления Playwright до последней версии выполните pip install --upgrade playwright и затем playwright install для обновления браузеров.
