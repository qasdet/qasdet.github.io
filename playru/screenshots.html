Скриншоты (Screenshots)

Скриншоты страниц и элементов для визуального тестирования, отладки и документации.

Базовый скриншот страницы

from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch()
    page = browser.new_page()
    page.goto("https://example.com")
    page.screenshot(path="screenshot.png")
    browser.close()
Примечание: Путь в path указывается относительно рабочей директории. Можно передать абсолютный путь.

Полностраничный скриншот

page.screenshot(path="full.png", full_page=True)

Скриншот области (clip)

page.screenshot(path="clip.png",
    clip={"x": 10, "y": 20,
           "width": 200, "height": 100})

Скриншот элемента

loc = page.locator("h1")
loc.screenshot(path="element.png")

JPEG с качеством

page.screenshot(path="photo.jpg",
    type="jpeg", quality=80)

Прозрачный фон

page.screenshot(path="nobg.png",
    omit_background=True)

Скриншот в буфер (bytes)

screenshot_bytes = page.screenshot()
Совет: Если не указывать path, page.screenshot() возвращает bytes. Удобно для сравнения пикселей.

Сравнение с эталоном

to_have_screenshot

from playwright.sync_api import expect

expect(page).to_have_screenshot("etalon.png")

При первом запуске эталон создаётся автоматически. При последующих — происходит сравнение.

Сравнение скриншота элемента

expect(locator).to_have_screenshot("element-etalon.png")

Отключение анимаций

page.screenshot(path="no-anim.png",
    animations="disabled")

Маскирование элементов

page.screenshot(path="masked.png",
    mask=[page.locator(".ads")],
    mask_color="red")
Важно: Маскирование изменяет только скриншот, не затрагивая реальную DOM-страницу.
