Видео (Video Recording)

Запись видео прохождения тестов для последующего анализа.

Включение записи видео

Видео включается при создании контекста через параметр record_video_dir.

from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch()
    context = browser.new_context(
        record_video_dir="videos/"
    )
    page = context.new_page()
    page.goto("https://example.com")
    context.close()
    browser.close()
Примечание: Видео сохраняется в формате WebM при закрытии контекста. Каждая страница получает отдельный файл.

Настройка размера видео

context = browser.new_context(
    record_video_dir="videos/",
    record_video_size={"width": 1280,
                        "height": 720}
)

Размер видео не может превышать 800x800 пикселей. Playwright подгоняет его под ограничение.

Путь к видео

context.close()
video_path = page.video.path()
print(video_path)

Сохранение копии

context.close()
page.video.save_as("output.webm")

Удаление видео

context.close()
page.video.delete()
Важно: Вызов delete() до закрытия контекста не даст эффекта. Видео должно быть сначала сохранено.

Видео в pytest

# Запись видео для каждого теста
pytest --video on

# Видео только при падении теста
pytest --video retain-on-failure

Пример фикстуры с видео:

import pytest

@pytest.fixture(scope="function")
def context(browser):
    ctx = browser.new_context(
        record_video_dir="videos/"
    )
    yield ctx
    ctx.close()

def test_example(context):
    page = context.new_page()
    page.goto("https://example.com")
    expect(page).to_have_title("Example")
Совет: Комбинируйте --video retain-on-failure с --tracing on для максимальной информативности при отладке.
