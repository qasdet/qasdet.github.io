ARIA-снимки (Aria Snapshots)

Текстовое представление accessibility-дерева страницы для проверки семантической структуры.

Что такое ARIA-снимок?

Снимок строится на основе ARIA-ролей и доступных имён элементов. Формат напоминает YAML — иерархическое дерево узлов.

- text "Example Domain"
- link "More information"

to_match_aria_snapshot

from playwright.sync_api import expect

expect(page).to_match_aria_snapshot("""
- heading "Example Domain" [level=1]
- paragraph "This domain is for use..."
- link "More information"
""")

Сравнение элемента

expect(
    page.locator("nav")
).to_match_aria_snapshot("""
- list
  - listitem
    - link "Home"
  - listitem
    - link "About"
""")

Формат снимка

Узел: роль "имя" [атрибуты]

Роли: button, heading, link, textbox, list, listitem, img, paragraph, text, checkbox, radio, combobox, slider, progressbar, tab, tabpanel, dialog, alert, tooltip и другие.

- button "Отправить"
- heading "Заголовок" [level=2]
- link "Читать далее"
- textbox "Имя" [disabled]
- img "Логотип"

Атрибуты узлов

АтрибутОписание
[checked]Флажок отмечен
[disabled]Элемент неактивен
[level=N]Уровень заголовка
[expanded]Раскрытый элемент
[pressed]Нажатая кнопка
Совет: Можно опускать атрибуты и дочерние узлы — снимок проверит только указанные. Это даёт частичное совпадение.

Частичное и строгое совпадение

Частичное совпадение (по умолчанию)

# Проверяет только наличие link "Home" среди прочих
expect(page.locator("nav")).to_match_aria_snapshot("""
- list
  - listitem
    - link "Home"
""")

Строгое совпадение (children: equal)

expect(page).to_match_aria_snapshot("""
- list [children: equal]
  - listitem
    - link "Home"
  - listitem
    - link "About"
""")

Глубокое равенство (deep-equal)

expect(page).to_match_aria_snapshot("""
- list [deep-equal]
  - listitem
    - link "Home"
""")

Regex в имени

expect(page).to_match_aria_snapshot("""
- heading /Добро пожаловать|Welcome/
- link "/page/\\d+/"
""")
Примечание: Обратную косую черту в regex нужно экранировать внутри формата снимка.

Получение снимка

snapshot_str = page.aria_snapshot()
print(snapshot_str)

snapshot = page.locator("main").aria_snapshot()
print(snapshot)

Assertion vs Snapshot-тестирование

to_match_aria_snapshotSnapshot-файлы
Снимок встроен в кодСнимок в отдельном файле (.snap)
Подходит для небольших фрагментовУдобен для больших деревьев
Частичное совпадение включеноСтрогое совпадение по умолчанию
Обновляется вручнуюОбновляется флагом --update-snapshots
# Snapshot-файл: __snapshots__/menu.aria
# - list
#   - listitem
#     - link "Home"
Важно: ARIA-снимки зависят от реализации accessibility браузером. Разные браузеры могут возвращать разное дерево для одной и той же страницы.
