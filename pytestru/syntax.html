Основы синтаксиса

Отступы и блоки кода

Python использует отступы (пробелы или табуляцию) для обозначения блоков кода. Стандарт — 4 пробела.

if age >= 18:  # условие
    print("Совершеннолетний")  # вывод в консоль
    can_vote = True  # логическое значение
else:  # иначе
    print("Несовершеннолетний")  # вывод в консоль
    can_vote = False  # логическое значение
