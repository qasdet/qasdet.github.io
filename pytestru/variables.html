Переменные и типы данных

Переменные

Python — язык с динамической типизацией. Тип переменной определяется автоматически.

name = "Анна"        # str
age = 30              # int
height = 1.75         # float
is_student = False    # bool
hobbies = None        # NoneType

# Множественное присваивание
a, b, c = 1, 2, 3

# Одно значение — нескольким переменным
x = y = z = 0  # присваивание

Проверка и приведение типов

# Проверка типа
type(42)           # 
isinstance(42, int)  # True

# Приведение (кастинг)
int("123")         # 123
str(42)            # "42"
float("3.14")      # 3.14
bool(1)            # True
bool(0)            # False
bool("")           # False
bool([])           # False

# Что считается False:
# 0, 0.0, "", [], (), {}, set(), None, False

Ничего (None)

result = None  # пустое значение
if result is None:  # условие
    print("Результата нет")  # вывод в консоль
Python различает is (проверка тождества) и == (проверка равенства). Для сравнения с None всегда используйте is None.
